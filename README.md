# Watson's Practice Java Project
## Introduction

This is a project dedicated mainly to solidify Java concepts, from scanners and
user inputs to enhanced switches and threads. It is a living, changing project
that will hopefully showcase the thought process from first practice classes
to a full-fledged program/application. As of May 28, 2026, the project has begun
to focus on a mock "banking" system for the households of Dune, Frank Herbert's 
science fiction novel. This may change the ultimate shape of the project. 

Branches are retained until they are successfully merged into main, at which point 
they are deleted.
